# 🎵 Bulgaria Music TV M3U8 Scraper

This open-source tool automatically scrapes Bulgarian and international music TV channels from [gledaitv.fan](https://www.gledaitv.fan/) and [bg-gledai.video](https://www.bg-gledai.video/) to generate a dynamic, updated `.m3u` IPTV playlist.

## 🚀 Live Playlist URL
If you just want to use the generated playlist inside VLC Player or an IPTV app, copy the URL below:
```text
https://githubusercontent.com
```

## 🛠️ Installation & Usage

1. Clone this repository:
   ```bash
   git clone https://github.com
   cd bg-music-iptv
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the script to fetch channels and generate the `.m3u` file:
   ```bash
   python scraper.py
   ```

## 📝 Disclaimer
This project is for **educational purposes only**. The scraper only aggregates publicly accessible links over the internet. We do not host any stream or video media.
